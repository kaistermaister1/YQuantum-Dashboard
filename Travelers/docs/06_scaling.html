C · The Block-Diagonal Advantage
1
The full QUBO is 200 x 200 — looks like a 200-qubit nightmare.
2
But zoom in: it has 10 blocks of 20 x 20 on the diagonal. Off-diagonal? All zeros.
3
These blocks are INDEPENDENT! Each package is its own sub-problem.
4
Instead of one 200-variable logical problem register, work with ten 20-variable package-local subproblems.
5
The block structure reduces the logical problem register from 200 variables to 20 variables per package. Whether the full algorithm fits still depends on ancillas, syndrome qubits, and the hardware model.
You do not automatically need a 200-qubit problem register.
Under exact block decomposition, the package-local logical size is 20. Total hardware needs still depend on the algorithm and any ancilla/syndrome overhead.
