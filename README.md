# YQuantum-Dashboard

- **`QGarsDashboard/`** — Next.js app (run `npm install` and `npm run dev` inside that directory).
- **`bundle_qubo/`** — Travelers-track **insurance bundling QUBO** (`qubo_block.py`), tests, and **Raylib `qubo_vis/`** (everyone gets this via `git pull`). Put LTM CSVs in a sibling **`Travelers/docs/data/YQH26_data/`** clone or pass `--data-dir`.
- **`DESIGN_SYSTEM_FOR_AGENTS.md`** — shared styling brief for dashboards and tools.

A full local clone of the official challenge repo can still live in **`Travelers/`** at the workspace root; that path remains gitignored here so two remotes do not collide. Use **`bundle_qubo/`** as the shared QGars copy on GitHub.
